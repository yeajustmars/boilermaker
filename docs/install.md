# Install Boilermaker

> TL;DR; Download the latest (CLI) binary from [https://github.com/yeajustmars/boilermaker/releases](https://github.com/yeajustmars/boilermaker/releases).


## Supported Platforms

Boilermaker is available for Linux and macOS, x86_64 and aarch64.

> _That should cover almost any Linux distro, Asahi, Intel Macs, as well as Apple Silicon._

## Download the latest release

As of writing (and during beta testing) the easiest way to install Boilermaker is to download the latest CLI binary release from the [GitHub releases page](https://github.com/yeajustmars/boilermaker/releases) or use the provided BASH script from there.

## Package Managers

There are near-future plans to have packages for many popular package managers, but we're just not there yet!
