# Install Boilermaker

> TL;DR; Download the latest binary from [https://github.com/yeajustmars/boilermaker/releases](https://github.com/yeajustmars/boilermaker/releases).


## Boilermaker is available for Linux and macOS, x86_64 and aarch64.

> _That should cover almost any Linux distro, Asahi and Intel Macs as well as Apple Silicon._

As of writing (and during beta testing) the easiest way to install Boilermaker is to download
the latest binary release from the [GitHub releases page](https://github.com/yeajustmars/boilermaker/releases)
or use the provided BASH script from there.

There are near-future plans to have packages for many popular package managers, but we're just
not there yet!

> If you'd like to get involved, hit us up on [GitHub](https://github.com/yeajustmars/boilermaker/discussions).

Next, see the [Getting Started](/docs/getting-started) guide to see how to use Boilermaker!
