Welcome to Boilermaker!

Making boilerplate more sane!

About

Boilermaker is a language-agnostic project template system for getting code projects up-and-running fast.

Yea, that's a mouthful. It just means you can write templates for one or many files, for any language you want!

  • It comes with a CLI and a desktop UI, and both are 100% compatible user interfaces, so... dealer's choice.
  • It's designed to be easy to create your own ready-to-run templates.
  • It's built in Rust so it's pretty fast, too.

Any official templates are guaranteed to be runnable, out of the box. This means you can get started in seconds, not hours!

Getting Started

First things first, install Boilermaker.

Hello World in Rust

        boil install https://github.com/yeajustmars/boilermaker \
    --subdir examples/hello-world \
    --lang rust \
    --name hello-world-rs

boil new hello-world-rs -d /tmp

cd /tmp/hello-world-rs

cargo run

From here, check out the documentation to learn more about Boilermaker and how to use it!
