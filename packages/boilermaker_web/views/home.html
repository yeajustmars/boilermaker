About

Boilermaker is a language-agnostic project template system for getting code projects up-and-running fast.

It comes with a CLI and a desktop UI, and both are 100% compatible user interfaces, so dealer's choice.

It's built in Rust so it's pretty fast, too.

Boilermaker is designed to be easy to create and use ready-to-run templates. Any official templates are guaranteed (minus finding a bug) to be ready to execute, out of the box. This means you can get started in seconds, not hours.

Getting Started

First things first, install Boilermaker.

Hello World in Rust

      boil install https://github.com/yeajustmars/boilermaker \
  --subdir examples/hello-world
  --name hello-world-rs

boil new hello-world-rs -d /tmp

cd /tmp/hello-world-rs

cargo run
