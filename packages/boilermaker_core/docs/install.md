# Install Boilermaker

## Supported Platforms

Boilermaker is available for Linux and macOS, x86_64 and aarch64.

> _That should cover almost any Linux distro, including Asahi, plus Intel Macs, as well as Apple Silicon._

## macOS

You can install Boilermaker on macOS using Homebrew:

```bash
brew tap yeajustmars/tap
brew install boilermaker
```

## Binaries

Download the latest CLI binary release from the [GitHub releases page](https://github.com/yeajustmars/boilermaker/releases) or use the provided BASH script from there.
